# Amazon Clone (HTML & CSS)

## 📌 Project Overview

This project is a **front-end clone of the Amazon homepage** created using **HTML and CSS**.
The goal of this project is to practice **web design, layout techniques by recreating the UI of the Amazon website.

It focuses on building the **visual structure of the website**, including the navigation bar, product sections, and footer.

---

## 🚀 Features

* Amazon-like **navigation bar**
* **Search bar UI**
* **Hero section/banner**
* **Product category boxes**
* **Footer section similar to Amazon**
* Clean and organized **HTML & CSS structure**

---

## 🛠 Technologies Used

* HTML5
* CSS3
* Flexbox

---



This project replicates the **basic UI of the Amazon homepage** including:

* Header with logo and search bar
* Product category grid
* Promotional banner
* Footer with links

Through this project I practiced:

* Building structured HTML layouts
* Styling using CSS
* Using Flexbox for layout alignment
* Creating reusable UI components
* Designing real-world website interfaces
